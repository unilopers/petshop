# petshop
O ERP PetShop é um sistema integrado de gestão voltado para pet shops de pequeno e médio porte. Seu objetivo é centralizar as principais operações do negócio, como o controle de animais, clientes, produtos, serviços e vendas, em uma única plataforma.
Com esse sistema, o gestor pode registrar atendimentos, agendar serviços como banho e tosa, controlar o estoque de produtos e acompanhar o faturamento do estabelecimento de forma prática e eficiente
